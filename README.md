
  # Mohammed Faheem FS Portfolio

  Business analyst portfolio built with React, Vite, and Tailwind CSS.

  The site is structured around honest concept case studies, recruiter-friendly messaging, and working resume/contact actions.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run build` to create a production build.
